Searches through an arbitrarily related model on its name field.

Note that this works only if

  • the related model has a field ‘name’

  • the filter using this method has the same name as the field of the model on which the filter is applied.

    (E.g. the field ‘profession’ on a person relates to another model: the professiontype. Here the filter on a person must also be called ‘profession’ as the field ‘profession’ exists within the person model and is then used to search in. Using this example of professions, such a lookup would be generated: Person.objects.filter(profession__name__… ) )
